# 时机 - 个体化时间能量决策助手

一个基于 Vue 3 的时间能量管理应用，帮助用户根据24小时能量分布做出更好的决策。

## 功能特性

- 🕐 **24小时能量分布图表** - 可视化展示每小时的能量水平
- 📊 **实时北京时间检测** - 自动显示当前时间对应的能量状态
- 🎯 **个性化行为建议** - 根据选择的行为类型提供定制化建议
- 📈 **能量分析报告** - 展示最佳和最差时间段
- 👤 **个人资料管理** - 五行偏向和偏好设置
- 📱 **响应式设计** - 适配桌面和移动端

## 技术栈

- **Vue 3** - 使用 Composition API
- **Pinia** - 状态管理
- **Vite** - 构建工具
- **原生CSS** - 样式设计

## 使用说明

1. **今日页面** - 查看24小时能量分布，选择时间段和计划行为
2. **分析页面** - 查看能量分析和最佳/最差时间段
3. **个人页面** - 查看和管理个人资料设置

## 特色功能

### 智能时间检测
应用会自动检测当前北京时间，并在图表中高亮显示当前小时。

### 动态行为推荐
根据选择的行为类型（工作、会议、运动等），系统会在推荐和避免行为列表中高亮显示相关建议。

### 响应式图表交互
- 悬停效果：鼠标悬停时显示透明度变化
- 选中状态：点击小时条会放大并改变颜色
- 当前时间：红色边框标示当前小时
